## Technologies Used
* Vanilla JS (ternary, class, constructor)
* Real DOM manipulation (preventDefault, rendering the chat per room)
* Array Manipulation methods (.forEach, setTimeout)
* Firebase (database, snapshot methods like .docChanges)
* Bootstrap 
* Date fns
